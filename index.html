Это запасная страница репозитория

Нормальная витрина — статический сайт из папки site/, который публикует workflow Deploy Portfolio site (GitHub Pages). Готовый сайт целиком заменяет то, что отдаётся с github.io, поэтому после успешного деплоя вы не должны видеть этот файл.

Если вы долго видите README или эту страницу вместо визитки:

  1. Settings → Pages → источник: GitHub Actions (не «Deploy from a branch»).
  2. Actions → Deploy Portfolio site (GitHub Pages) → последний прогон зелёный.
  3. Полная перезагрузка без кэша: Ctrl+F5 или другой браузер / режим инкогнито.
