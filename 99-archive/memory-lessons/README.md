# Memory lessons: варианты памяти Telegram-бота

**Что это:** три варианта одного сценария — **короткая**, **длинная** и **гибридная** память в диалоге (отдельные скрипты-боты).

**Файлы:**

| Файл | Смысл |
|------|--------|
| `bot_short_memory.py` | краткосрочный контекст |
| `bot_long_memory.py` | устойчивое хранение контекста |
| `bot_hybrid_memory.py` | комбинированный подход |

**Стек:** Python, aiogram/telegram (см. `requirements.txt`), `.env.example` для ключей.

**Запуск:** скопируйте `.env.example` в `.env`, заполните токен и API-ключи, затем `python bot_<variant>_memory.py` для нужного варианта.
