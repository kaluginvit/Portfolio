# Prompting case: сравнение промптов

**Что это:** сценарии сравнения промптов к GPT-4o: скрипты запуска и **готовые артефакты** ответов в `.md` и `.json`.

**Стек:** Python (`prompt_chat.py`, `prompt_compare.py`), результаты экспериментов в репозитории.

**Как использовать:** изучите файлы `01_*`, `02_*`, `03_*` — это результаты прогонов; скрипты помогают воспроизвести сравнение на своих ключах.

**Секреты:** не коммитьте API-ключи; задавайте через переменные окружения при локальном запуске скриптов.
