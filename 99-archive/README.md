# Архив

Учебные уроки, черновики и эксперименты: они остаются для истории и не должны конкурировать по «витрине» с основными кейсами в папках `01-` … `05-`.

Подпапки — самостоятельные мини-проекты (API, PostgreSQL, Docker, LangChain, MCP, автономные агенты и т.д.). У многих есть свой `README.md`; если его нет, ориентируйтесь по имени папки и исходному коду.
