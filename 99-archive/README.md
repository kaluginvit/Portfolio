# Архив — 15 проектов с потенциалом доводки

После пересборки портфолио в архиве остались **15 проектов** с шансом перерасти в самостоятельный кейс или утилиту. Семь самых сильных проектов отсюда уже подняты в направления `02-automation`, `03-ai-products` и `04-web`. Учебные обрывки и дубли удалены из репозитория.

Архив — не «свалка истории», а **песочница на доводку**: каждая папка ниже либо доделывается до полноценного продукта, либо служит методическим артефактом с рабочим кодом.

## Зачем такое деление

Архив с префиксом `99-` уезжает в самый низ файлового дерева на GitHub UI и в файловом менеджере. Витрина (папки `01-` … `05-`) при этом остаётся плотной — рекрутёр или клиент видит сначала продакшен-кейсы, а в архив заглядывает уже после, **по своей воле**, как в технический «бэклог».
