# AI automation lessons

**Что это:** пайплайн «данные → обработка через API → отчёт клиенту», в т.ч. генерация PDF и блок `client_report/` (ИИ-обработка секций, стоимость, PDF).

**Стек:** Python; точки входа `main.py`, `api.py`, `telegram_bot.py`; пакет `client_report/`.

**Запуск:** создайте `.env` по `.env.example` в корне (если есть), установите зависимости и запускайте нужный модуль — см. импорты в `main.py`.

**Назначение:** архивный учебный код с потенциалом вынести в отдельный продуктовый репозиторий после рефакторинга.
