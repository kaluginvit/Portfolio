# Метрики продукта (практика)

**Что это:** набор разборов продуктовых метрик и отчётности: воронки, сегментация, реклама, Яндекс.Метрика.

**Стек:** текстовые разборы в Markdown (GA, Яндекс.Метрика и др.).

**Как пользоваться:** открывайте файлы `.md` в репозитории или в превью GitHub — это конспекты/отчёты без отдельного кода запуска.

**Состав:** см. файлы `Практика_метрики_продукта.md`, `Воронка_конверсий_GA.md`, `Сегментация_пользователей_GA.md`, `Анализ_рекламной_кампании_GA.md`, `Отчёт_Яндекс_Метрика.md`.
