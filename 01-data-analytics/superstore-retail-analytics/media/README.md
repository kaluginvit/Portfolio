# Медиа для портфолио

Рекомендуемые файлы:

- `dashboard-kpi.png` — верхняя часть дашборда с KPI.
- `map-or-region.png` — карта или региональный срез.
- `presentation-slide.png` — один слайд из `superstore_presentation.html`.

После добавления вставьте в корневой `README.md` превью, например:

```markdown
![Дашборд](media/dashboard-kpi.png)
```
