# Анализ тестового швейного сценария (Wildberries)

**Что это:** EDA по еженедельным отчётам WB: объединение CSV, сводные метрики, визуализации в ноутбуках.

**Стек:** Python, Jupyter, pandas; `requirements.txt` при необходимости добавьте сами (`pandas`, `openpyxl`, `matplotlib` / `plotly` по ноутбуку).

**Как запустить:** открыть `wb_report_eda.ipynb` или `wb_weekly_report_eda.ipynb`; для объединения отчётов используйте `merge_reports.py`, `export_summary_to_csv.py`, `set_csv_for_analysis.py` по подсказкам внутри кода.

**Артефакты:** CSV с маркетплейса (обезличить при публикации), HTML-графики `BDDS_DDS_*.html`.
