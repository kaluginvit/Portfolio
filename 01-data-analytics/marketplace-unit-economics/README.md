# Юнит-экономика маркетплейса

**Что это:** расчёт и визуализация юнит-экономики и когорт продавцов на маркетплейсе.

**Стек:** Python, Jupyter, pandas; презентация в `presentation.html`.

**Как запустить:** открыть `marketplace_unit_economics.ipynb`; CSV с данными лежат в папке (когорты, продавцы).

**Артефакты:** ноутбук, HTML-презентация, исходные CSV для воспроизведения расчётов.
