# Python для бизнес-аналитики

**Что это:** учебные ноутбуки по Python/pandas с типовыми датасетами (заказы, визиты, покупки и др.).

**Стек:** Python, Jupyter, pandas; CSV во вложении.

**Как запустить:** установить `jupyter`, `pandas` (см. типичный набор в других проектах портфолио), открыть нужный `.ipynb`.

**Структура:** ноутбуки с номерами уроков и тем (`lesson_1_python.ipynb`, `python_3.ipynb` и т.д.) + файлы данных `.csv`.
