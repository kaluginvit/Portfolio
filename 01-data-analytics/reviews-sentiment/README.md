# Анализ отзывов (тональность)

**Что это:** ноутбук с анализом отзывов (на примере продукта из датасета).

**Стек:** Python, Jupyter; зависимости — `requirements.txt`.

**Как запустить:**

```bash
pip install -r requirements.txt
jupyter notebook yandex_station_reviews_analysis.ipynb
```

**Примечание:** при публикации портфолио не включайте персональные данные в CSV без обезличивания.
