# Портфолио

Репозиторий с проектами по **продуктовой аналитике**, **автоматизации процессов**, **ИИ-продуктам** (боты, RAG, MCP), **веб-приложениям** и **ИИ-консалтингу**. Проекты сгруппированы по тематическим папкам с префиксами `01-` … `99-`, чтобы навигация на GitHub была предсказуемой.

## Направления

| Папка | Содержание |
|--------|------------|
| [`01-data-analytics`](./01-data-analytics/) | SQL, Python, A/B-тесты, отчёты, дашборды |
| [`02-automation`](./02-automation/) | n8n, скрипты, API-сервисы, парсинг, интеграции |
| [`03-ai-products`](./03-ai-products/) | Telegram-боты, RAG, агенты, MCP-серверы |
| [`04-web`](./04-web/) | Сайты и full-stack (Next.js, Angular и др.) |
| [`05-ai-consulting`](./05-ai-consulting/) | Стратегии и материалы по внедрению ИИ |
| [`99-archive`](./99-archive/) | Учебные артефакты и эксперименты |

На GitHub: пути вида `https://github.com/kaluginvit-svg/Portfolio/tree/main/<папка>/`.
