# Локальный транскрибатор (faster-whisper)

**Что это:** CLI для **офлайн-транскрибации** аудио через **faster-whisper** (модели Hugging Face кэшируются локально).

**Стек:** Python 3.10+, `faster-whisper`, для mp3/m4a нужен **ffmpeg** в `PATH`.

**Статус:** рабочий утилитарный скрипт; подходит как демонстрация пайплайна «аудио → текст» без облачных API.

---

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python transcribe.py
```

Интерактивно скрипт предложит файл из папки `WAV/` или передайте путь к файлу аргументом (см. `python transcribe.py --help`).

Первый запуск скачает выбранную модель в кэш Hugging Face (`~/.cache/huggingface`).

---

## Пример артефакта

В [`WAV/`](./WAV/) лежат примеры `.txt` (транскрипты) рядом с тематикой занятий — можно использовать как **образец выхода** (без публикации исходного аудио при ограничениях).

Исходные **`.mp4` / крупные медиа** в репозиторий не кладутся (лимит GitHub 100 MB) — держите их **только локально** или в облаке; в git игнорируются через корневой `.gitignore`.

---

## Ограничения

- Качество и скорость зависят от модели и GPU/CPU.  
- Длинные файлы — следите за памятью.  
- Язык и модель задаются в `transcribe.py` (по умолчанию смотрите код).
